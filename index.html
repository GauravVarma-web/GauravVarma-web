Travel Planner Agent

Project Overview

This Tool-Based AI agent helps users plan travel by providing weather information, flight details, and attraction recommendations for various destinations.

Technical Implementation

The agent is built using the ReAct (Reasoning + Acting) pattern, which enables it to think step-by-step about a problem, take actions to gather information, and provide reasoned responses.

Key Features

  • Weather information for destinations
  • Flight time and price estimates
  • Local attraction recommendations
  • Multi-turn conversations
  • Reasoning capabilities

Technologies Used

Python, OpenAI API, ReAct pattern, JSON parsing, and a modular function architecture that can be extended to other domains.

RAG Marketing Agent

Project Overview

This Retrieval Augmented Generation (RAG) agent provides intelligent marketing insights by leveraging your marketing documents and combining them with advanced language model capabilities.

Technical Implementation

The agent uses vector embeddings to store and retrieve information from marketing documents, connecting this retrieval system with language models to generate contextually relevant and accurate responses.

Key Features

  • Document processing and indexing
  • Semantic search capabilities
  • Context-aware responses
  • Source attribution for transparency
  • Support for multiple document types

Technologies Used

LangChain, ChromaDB vector database, OpenAI embeddings and completions, Python, and document processing libraries.
