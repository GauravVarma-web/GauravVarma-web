Travel Planner Agent

Project Overview

This AI agent helps users plan travel by providing weather information, flight details, and attraction recommendations for various destinations.

Technical Implementation

The agent is built using the ReAct (Reasoning + Acting) pattern, which enables it to think step-by-step about a problem, take actions to gather information, and provide reasoned responses.

Key Features

  • Weather information for destinations
  • Flight time and price estimates
  • Local attraction recommendations
  • Multi-turn conversations
  • Reasoning capabilities

Technologies Used

Python, OpenAI API, ReAct pattern, JSON parsing, and a modular function architecture that can be extended to other domains.
